Server Setup
------------------------------------
To setup the IP address and port that the server will be serving from, modify the TCP_IP/UDP_IP and TCP_PORT/UDP_PORT variables in Server_TCP.py or Server_UDP.py

Launch server using "python3 Server_TCP.py" or "python3 Server_UDP.py"
 

Client Application
------------------------------------
Launch client application by using "python3 Client_TCP.py" or "python3 Client_UDP.py"

Application will prompt you for the server's IP address and port that you wish to send the message to. Next you will be prompted to enter your message for the server. Server will respond appropriately.
